## setup
Run `setup.sh`.
When everything installed correctly, run:
```bash
bombadil install $HOME/.dotfiles/bombadil.toml
```

To link these dotfiles in dark mode, run:
```bash
$ bombadil link -p sway_d
```

light mode would be:
```bash
$ bombadil link -p sway_l
```
